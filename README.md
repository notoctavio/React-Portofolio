# Octavio-Daniel Vizaru — Portfolio

Modern personal portfolio built with **React, TypeScript, Vite, and Tailwind CSS**.

**Live:** [octaviodanielvizaru.xyz](https://www.octaviodanielvizaru.xyz)

## Stack

- React 19
- TypeScript
- Vite 8
- Tailwind CSS 4
- pnpm

## Run Locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## Customize

- Content and links: `src/config/index.ts`
- UI components: `src/components/*`
- Global styles/animations: `src/styles/global.css`
- Metadata + favicon: `index.html`
- Static assets + CV: `public/*`

## Deploy

Deploy the `dist/` folder (Vercel/Netlify/static hosting).
