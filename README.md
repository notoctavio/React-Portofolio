## Features

- Smooth hero intro and polished section transitions
- Responsive, content-driven layout
- Dark/light theme toggle
- Contact popover with email actions
- SEO metadata + analytics integrations

## Tech Stack

- React 19
- TypeScript
- Vite 8
- Tailwind CSS 4
- pnpm

## Quick Start

```bash
pnpm install
pnpm dev
```

Production build:

```bash
pnpm build
pnpm preview
```

## Customization

- **Content:** `src/config/index.ts`
- **Components/UI:** `src/components/*`
- **Global styles:** `src/styles/global.css`
- **Metadata/Favicon:** `index.html`
- **Static assets/CV:** `public/*`

## Deploy

Deploy the `dist/` output to Vercel, Netlify, or any static host.

- Install command: `pnpm install`
- Build command: `pnpm build`
- Output directory: `dist`
